Git & Github

> What is VCS?

VCS means Version Control System.A VCS manages and tracks changes to documents, files and other collections of information.Its very usefull for Web Developers. A developer can work with another developer with VCS. Example: Git.

> What is Git?

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

> What is GitHub?

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. This tutorial teaches you GitHub essentials like repositories, branches, commits, and pull requests.
